First of all

  • Turn-on mic and video
  • Turn your speakers off (minimise volume) to get not disturbed by background noise.

About the exam

  • There are 30 MCQ questions (15 Theory, 15 R Programming)
  • Each question has 4 answer options
  • Each option is either true or false.
  • By marking you select yes if the answer is true in your opinion and No if it is false
  • Third option: If you are unsure, you can skip the answer option (see later)

Exercise: Technical Test

Execute the following steps

  1. Login to the ilearn platform
  2. Open the Exam Technical Test file.
  3. Try if you can work with the file. Three possibilities:
    • Marking it digitally with the tools of your pdf reader
    • Using the forms of the pdf (potentially risky!)
    • Write answers to sheet of paper that you send in afterwards
  4. Save the file and check if it worked

Evaluation

  • For each correct selection (yes/no) you get a point
  • For each wrong selection 1/3 point is deducted (3 wrongly marked -> 1 point off)
  • Not selecting an option: no point but also no deduction

Recommendation / Tips

  • At the beginning: Briefly browse the exam.
  • Maybe start with part 2: It probably will be faster (and you have more time for part 1).
